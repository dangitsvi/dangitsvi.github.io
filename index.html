About

My journey of becoming a developer

  • 2009-2011

    A Chef Emerges

    My life goal after graduating highschool was to open a line of Vietnamese Cuisine food trucks so I started by enrolling to Seattle Culinary Academy. While in school I worked multiple restaurant jobs including spending a year at Seattle's Lark Restaurant as a cold-line cook.

  • 2012-2014

    To Business, or not to Business

    With entrepreneurship in mind, I transferred into the University of Washington's business school to sharpen my business acumen. During this time, I also took an introductory programming class. This was my first taste of programming and I was quickly hooked.

  • January 2015

    My First App

    While looking for work, I wanted to try making an application using the basic Java I had learned. My brother, an android developer, helped set up my development environment and got me started. I quickly become engrossed with this project and spent all day and night coding for the next couple of months. I published the app March 2015.

  • July 2015

    The Fellowship of the Code

    After coding my first app, I knew I wanted to be a programmer so I applied to Code Fellows' Full-Stack JavaScript Bootcamp. I practicly lived in the Code Fellows' class rooms, doing nothing else but learning code for two months. Now I'm ready to put my skills to the test!
